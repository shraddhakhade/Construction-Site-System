# Construction Site Management System

Construction Site Management System is a desktop application developed using C# and .NET Framework to simplify the management of construction projects. The system provides a centralized platform for maintaining information related to construction sites, workers, suppliers, materials, purchase orders, payments, and project allocations.

The application enables administrators to efficiently manage site operations by storing and retrieving project-related data, tracking material usage, allocating workers and materials to specific sites, managing supplier and customer payments, and generating detailed reports.

## Key Features

* Material Management

  * Add, update, delete, and view material details.

* Supplier Management

  * Maintain supplier information and payment records.

* Worker Management

  * Store worker details and manage worker payments.

* Site Management

  * Maintain construction site information and project details.

* Enquiry Management

  * Record and track customer enquiries.

* Purchase Order Management

  * Create and manage material purchase orders.

* Site Allocation

  * Allocate workers and resources to construction sites.

* Material Allocation

  * Assign materials to specific project sites.

* Payment Management

  * Manage supplier, worker, and customer payments.

* Report Generation

  * Generate reports related to workers, materials, suppliers, payments, and site allocations.

## Technologies Used

* C#
* .NET Framework
* Windows Forms
* SQL Database

## Objective

The objective of this project is to reduce manual paperwork, improve data organization, and enhance the efficiency of construction project management through a user-friendly desktop application.
